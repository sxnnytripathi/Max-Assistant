# MaxAssistant 
## About
Max is a Virtual Assistant which uses complex machine learning algorithms to communicate and perform tasks.

## Requirements

```bash
pip install pyttsx3
pip install speechRecognition
pip install pyaudio
pip install wikipedia
pip install pil
pip install pygame
```
